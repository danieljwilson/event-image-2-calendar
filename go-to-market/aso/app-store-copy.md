# App Store Optimization (ASO)

## Current Copy

The canonical App Store submission copy lives in [APP_STORE_METADATA.md](../../APP_STORE_METADATA.md). This document focuses on optimization strategy.

## Name & Subtitle

**Current:**
- Name: "Event Snap"
- Subtitle: "Photos to calendar, instantly"

**Optimized alternative (more keyword-rich):**
- Name: "Event Snap — Photo to Calendar"
- Subtitle: "Scan posters, flyers & links"

The optimized version adds "Photo", "Calendar", "Scan", "posters", "flyers", and "links" — all high-intent search terms — while staying under the 30-character subtitle limit.

## Keywords (100 characters max)

**Current:** `event, poster, calendar, photo, extract, flyer, schedule, ics, share, snap`

**Optimized options to test:**
- `event,poster,calendar,photo,flyer,scan,schedule,concert,extract,snap` (adds "concert", "scan")
- `event,poster,calendar,photo,flyer,scan,concert,festival,meetup,ics` (adds "festival", "meetup")

**Keywords NOT to waste on** (already in name/subtitle and thus indexed automatically):
- event, snap, photo, calendar, scan, posters, flyers, links

**High-value keywords to prioritize in the 100-char field:**
- extract, schedule, ics, concert, festival, meetup, reminder, OCR, capture, import

## Description Strategy

The first 3 lines are visible before "more" — they must hook.

**Current first line:** "Photograph an event poster or share a link, and Event Snap extracts the details..."

This is good — it's clear and descriptive. Could be punchier:

**Alternative:** "See an event poster? Snap a photo. Event Snap uses AI to extract the title, date, time, and venue — and adds it to your calendar in one tap."

**Design guidelines:**
- Clean, minimal frames (device mockup optional — Apple now allows with/without)
- Large, readable caption text above each screenshot
- Consistent color scheme (match app accent color)
- Show real-world content (actual posters, not placeholder text)

## App Preview Video

**Duration:** 15-30 seconds
**Flow:**
1. (0-3s) Person sees a concert poster on a wall
2. (3-8s) Opens Event Snap, snaps photo
3. (8-15s) Extraction animation, details appear
4. (15-20s) Tap "Add to Calendar" — success
5. (20-25s) Quick montage: share extension from Instagram, URL extraction, daily digest
6. (25-30s) Tagline: "Snap a poster. Get a calendar event."

**Audio:** Upbeat background music, no voiceover (App Store auto-plays muted)

## ASO Iteration Plan

- **Week 1-2 post-launch:** Baseline metrics (impressions, conversion rate, keyword rankings)
- **Month 1:** A/B test screenshots (Apple provides this in App Store Connect)
- **Month 2:** Keyword rotation based on Search Ads data (if running Apple Search Ads)
- **Ongoing:** Monitor keyword rankings, adjust quarterly
